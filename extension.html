  • Extension

Attackflow Extensions are IDE extensions that are installed from the marketplace of Visual Studio and Eclipse. They ensure real-time secure coding for development teams.


There are three types of extensions exists; first is for Eclipse IDE (all versions), second is for IntelliJ IDEA (all versions) and the last one is for Visual Studio IDE (2015, 2017, 2019 and 2022).

Eclipse IDE Extension

Eclipse

  • Eclipse Marketplace
  • Limited vulnerability definitions
  • Limited vulnerability documentations
  • All vulnerability definitions
  • All vulnerability documentations
  • Real-time Integration with Enterprise Edition

IntelliJ IDEA IDE Extension

IntelliJ Idea

  • Jetbrains Marketplace
  • Limited vulnerability definitions
  • Limited vulnerability documentations
  • All vulnerability definitions
  • All vulnerability documentations
  • Real-time Integration with Enterprise Edition

Visual Studio IDE Extension

Visual Studio

  • Visual Studio Marketplace
  • Limited vulnerability definitions
  • Limited vulnerability documentations
  • All vulnerability definitions
  • All vulnerability documentations
  • Real-time Integration with Enterprise Edition
